Eslint

Install eslint

Run on your project directory :

$ npm install eslint --save-dev

Install eslint-config-airbnb

  • Bare in mind that airbnb has 3 peer dependencies

    • eslint-plugin-import
    • eslint-plugin-jsx-a11y
    • eslint-plugin-react

    Install the correct versions

    check the versions of the dependencies airbnb needs by running this on your command line

    $ npm info "eslint-config-airbnb@latest" peerDependencies

    it will produce somthing like this

    { eslint: '^4.19.1 || ^5.3.0',
  'eslint-plugin-import': '^2.14.0',
  'eslint-plugin-jsx-a11y': '^6.1.1',
  'eslint-plugin-react': '^7.11.0' }

    once you know what the dependency versions are next thing to do is to install them by running this command

    $ npm install --save-dev eslint-config-airbnb eslint@^#.#.# eslint-plugin-jsx- a11y@^#.#.# eslint-plugin-import@^#.#.# eslint-plugin-react@^#.#.#

    Note that # means the version number of the dependency.

    If you encounter this error at some point

    npm ERR! No compatible version found: a11y@^6.1.1
npm ERR! Valid install targets:
npm ERR! 0.5.1, 0.5.0, 0.4.1, 0.4.0, 0.3.3, 0.3.2, 0.3.1, 0.3.0, 0.2.6, 0.2.5, >0.2.3, 0.2.2, 0.2.1, 0.2.0, 0.1.2, 0.1.1, 0.1.0

    Don't panic just change the eslint version to what version it is on your package.json

Install babel-eslint

We need this so eslint could process babel transformed code

$ npm install eslint babel-eslint --save-dev

Install react-native plugin for eslint

  • eslint-plugin-react-native
$ npm install --save-dev eslint-plugin-react-native

Configuring eslint

Once we completed the installation process we need to configure it to work with us

Create .eslintrc.js file on the root project tree

  • Once you created the file paste the following code to that file
module.exports = {
    "env": {
        "es6": true,
        "node": true,
        "react-native/react-native": true
    },
    "extends": "airbnb",
    "parser": "babel-eslint",
    "globals": {
        "Atomics": "readonly",
        "SharedArrayBuffer": "readonly"
    },
    "parserOptions": {
        "ecmaFeatures": {
            "jsx": true
        },
        "ecmaVersion": 2018,
        "sourceType": "module"
    },
    "plugins": [
        "react",
        "react-native"
    ],
    "rules": {
        "react/jsx-filename-extension": "off",
        "react/prefer-stateless-function": [0, { "ignorePureComponents": false }],
        "react/prop-types": ["error", { "ignore": ["navigation"] }],
        "quotes": [0, "double", { "avoidEscape": true }],
        "no-underscore-dangle":  ["error", { "allow": ["_navigator"] }],
        "arrow-parens": ["error", "as-needed"],
        "import/no-unresolved": 2,
        "no-console": "off",
        "max-len": 0,
        "react/forbid-prop-types": [0],
        "no-unused-expressions":  ["error", { "allowShortCircuit": false, "allowTernary": true }],
        'semi': [0, 'always'],
        "vars-on-top": 2,
        "require-await": 2,
        "no-await-in-loop": 2,
        "no-return-await": 2,
        "no-multi-spaces": ["error", { ignoreEOLComments: false }],
        "func-style": ["error", "declaration", { "allowArrowFunctions": true }],
        "react-native/no-inline-styles": 2,
        "react-native/no-color-literals": 2,
        "react-native/no-raw-text": 2,
    }
};
